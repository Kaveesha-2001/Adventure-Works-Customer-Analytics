# Adventure Works Customer Details
## Project Overview
This repository contains a Power BI project analyzing customer behavior and sales trends. The goal was to transform raw data into actionable business insights.

## Technical Skills Applied
- Data Cleaning: Used Power Query to handle missing values and format data.

- Data Modeling: Established relationships between customer, product, and sales tables.

- DAX Measures: Created calculated measures for Total Revenue and Order Targets.

- Advanced Visuals: Integrated Map visuals, Treemaps for age distribution, and Gauge charts for KPI tracking.

## Key Insights
- Revenue Analysis: Total revenue generated is $24.9M across 25k+ orders.

- Customer Profiling: Balanced sales distribution between Male (50%) and Female (49%) customers.

- Visualized specific product demand (e.g., Water Bottles and Touring Bikes).
